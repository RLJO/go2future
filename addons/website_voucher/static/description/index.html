Now offer Coupons & Vouchers to your Odoo Website Customers

Odoo Website Coupons & Vouchers module comes up with the functionality to create digital discount coupons for your customers in Odoo. The discount can be set as fixed or percent of the total cart amount.

Each coupon contains a unique code to validate it with the purchase. The customer just needs to enter the available coupon code to get the defined discount. The Odoo user can create general coupons and vouchers for all customers, or special coupons for specific customers.
